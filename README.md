## PSCsession Documentation By Psconfigurator Team

This module allows to manage a time session to ephemeral demo shops.


## Installation

1) Unzip and upload the module folder in your "modules" folder of your installation Prestashop.
2) Browse the "pscsession" at  Modules list and click on the “Install” button to install this Module.
3) Go to  Modules > Pscsession > Configure >  To configure the end date of the session
4) Go to localization / translations to translate the module to any language (IF NECESSARY)
